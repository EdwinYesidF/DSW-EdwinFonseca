Sitio web para el aprendizaje de los Flex Box y CSS Grid

El propósito general de esta pagina es dar a conocer las características y la usabilidad
de los Flex Box y CSS Gris en el Diseño de páginas web.

Contenido

La caja flexible, más comúnmente conocido como flexbox , es una herramienta
utilizada para crear diseños con CSS. como CSS Grid, que puede resolver
varios problemas de diseño. Flexbox es un diseño unidimensional que
coloca elementos tanto horizontal como verticalmente (pero no ambos).
Un elemento que usa flexbox como diseño se conoce como contenedor flexible , y
los elementos dentro de él son elementos flexibles .

CSS Grid Layout es un modelo de diseño que tiene la capacidad de controlar el tamaño
la posición de las cajas y su contenido. la diferencia con los flexbox , es que los flex
box estan orientados en un solo eje, en el CSS Grid el diseño de la cuadricula se optimiza para
diseños bidimensionales: aquellos en los que se desea alinear el contenido en ambas dimensiones.
